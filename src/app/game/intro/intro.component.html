GAME INTRO

Welcome to Angular Poker!

This is a poker game built using Angular 12 and Firebase. The game is Texas Hold'em with only one initial round of betting.

This game was made by myself and my mentor. Although the game is not fully funtional the base elements of the game are in place.

The game is scafolded correctly and has the correct game logic, all within the correct Angular components and services.
